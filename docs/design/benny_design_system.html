Quy tắc bắt buộc: khi tạo BẤT KỲ màn hình / component nào, luôn phải định nghĩa màu cho cả Light Mode và Dark Mode (mỗi token có cặp giá trị light/dark). Không hardcode màu — mọi background, surface, text, border, icon đều phải lấy từ token để tự đổi theo chế độ. Mỗi tính năng có Light Mode thì cũng phải có Dark Mode tương ứng.

🎨 Mới ở design (chưa build code): Snackbar có nút tắt · Controls (switch/checkbox/radio/appbar/list tile) · Profile (Hồ sơ) + nút Dark Mode · Status screens (→ đề xuất AppStatusView) · Dialog & Bottom sheet · password rule popover · Dark Mode đầy đủ (mục 06, bảng màu tối ở mục 01).
🌙 Dark Mode: cần thêm bộ token tối vào theme (ThemeState/AppColors) + cờ light/dark; lưu lựa chọn; toggle ở Profile. Hiện app mới chỉ có Light Mode.
